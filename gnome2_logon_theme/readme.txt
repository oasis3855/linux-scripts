ABOUT THIS FILE
      This package is Logon Theme for Gnome Desktop Manager on Linux.
      このパッケージは、GDM用ログオンテーマです。

HOW TO INSTALL
      Install from 'Login Screen' system setup dialog (Gnome menu -> System
      -> Administration -> Login Screen).
      GDMシステム設定の"ログオン画面"よりインストールします。

REFERENCES
      http://library.gnome.org/admin/gdm/stable/thememanual.html.en
